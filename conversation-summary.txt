Search criteria format:
- Use the same style as `--keep`, but with `--search` instead of `--keep`.
- Example:
  python emailtaxmenot.py --action clean --email you@example.com \
    --search --from sender@example.com --subject "newsletter" --body "unsubscribe"
- The criteria should be combined into a flat IMAP search list:
  ["FROM", value, "TO", value, "SUBJECT", value, "BODY", value]
- This effectively ANDs the fields in IMAP search.

```python
def parse_search_groups(tokens):
    groups = []
    current = None
    i = 0

    while i < len(tokens):
        token = tokens[i]

        if token == "--search":
            if current is not None:
                groups.append(current)
            current = {"criteria": {}}
            i += 1
            continue

        if current is None:
            raise ValueError("Expected --search before search-group options")

        if token in ("--from", "--to", "--subject", "--body"):
            key = token.lstrip("-")
            current["criteria"][key] = tokens[i + 1]
            i += 2
            continue

        raise ValueError(f"Unknown token: {token}")

    if current is not None:
        groups.append(current)

    return groups
```

```python
def build_imap_criteria(criteria):
```
